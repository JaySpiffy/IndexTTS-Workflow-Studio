# Frontend Debug Archive

This folder holds older frontend debugging aids that are no longer part of the shipped app surface.

## What Lives Here

- `html/`
  - one-off browser test pages and debug harnesses
- `docs/`
  - older frontend debugging notes and workflow writeups
- `legacy/`
  - superseded frontend files kept only for historical reference

These files are intentionally kept out of `frontend/` so the release-facing UI folder only contains the actual app.

If you use anything from here for troubleshooting, treat it as archival material and verify it against the current frontend before relying on it.
