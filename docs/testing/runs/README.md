# Test Run Records

Store completed manual smoke logs and release verification notes here.

Recommended naming:
- `YYYY-MM-DD-smoke.md`
- `YYYY-MM-DD-release.md`
- `YYYY-MM-DD-regression-<topic>.md`

Start from:
- [../test-run-log-template.md](../test-run-log-template.md)

Keep these records short and practical:
- what was tested
- what passed
- what failed
- what still needs follow-up
