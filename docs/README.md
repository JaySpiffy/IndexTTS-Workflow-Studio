# Docs Index

This folder holds supporting documentation that used to clutter the repo root.

## Sections

- `api/`
  - current API summary only
- `archive/`
  - superseded planning docs and older reference notes
- `deployment/`
  - deployment, stack-running, and backend-only runtime guidance
- `manual/`
  - tab-by-tab user manual with screenshots
- `project/`
  - release and parity tracking documents
  - known limitations and release readiness status
- `research/`
  - research notes, listening syntax, and quality-planning docs
  - includes the IndexTTS2 scripting playbook and reusable dialogue system prompt
  - includes `research/white-papers/` for deeper prompt packs, source outputs, and synthesis notes
- `testing/`
  - repeatable smoke-test packs and release checklists for the live app
- `timeline/`
  - current timeline pointer only
  - historical timeline planning docs live under `archive/timeline/`
