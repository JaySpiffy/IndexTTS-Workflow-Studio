# Draft to Take Audio Examples

These short examples were generated with Draft to Take and are included so visitors can hear the beta without running the full stack first.

- `script-canvas-mini-mix.mp3` - a short Script Canvas export with generated dialogue.
- `emotion-fear.mp3` - an emotion-vector sample using a fear-led delivery.
- `emotion-anger-disgust.mp3` - an emotion-vector sample mixing anger and disgust.

The examples are for product demonstration only. They do not include model weights or private source assets.
